InnerWhispers Email Dashboard

Use this central landing page to access the HR dashboard, email templates, status monitoring, and profile pages. The navigation is ordered top-to-bottom for easy workflow.

  • HR Dashboard Dashboard

    Open the main HR dashboard for candidate actions, bulk email, and quick send buttons.

  • Application Email Template

    Use this page to send or preview application received emails for interns.

  • Offer Letter Template

    Send offer letter emails and open the candidate-specific offer letter experience.

  • Email Status Logs

    View email logs, search by name or email, and monitor sent / failed / pending counts.

  • Profile Creation Template

    Preview the profile creation email and help new interns onboard quickly.

Tip: Update html/js/config.js with your Render backend URL before deployment. The dashboard will then call the backend endpoints automatically.
